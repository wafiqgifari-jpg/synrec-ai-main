# SYNREC — Movie Recommendation System
Capstone Project CC26-PSU255 | Coding Camp 2026 powered by DBS Foundation

## Description
SYNREC recommends movies based on synopsis similarity using Transfer Learning 
with Sentence-BERT and TensorFlow Functional API. No watch history needed — 
just describe the story you want.

## Tech Stack
- Sentence-BERT (all-mpnet-base-v2)
- TensorFlow Functional API
- Python, NumPy, Scikit-learn, FastAPI
